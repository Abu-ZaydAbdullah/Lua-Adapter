# Lua-Adapter
Use this wrapper- / adapter-class as an interface between Lua and C++.

Usage:
- see main.cpp

Compile: "g++ -std=c++0x lua_adapter.cpp main.cpp  -llua -ldl"



Requirements:
- g++/gcc compiler
- liblua5.2-dev
- liblua5.2-0 (or later)


You could also use it in C, with some minor changes.
